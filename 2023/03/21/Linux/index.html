Linux部分

操作系统也是一个软件，这个软件封装了很多的功能来操作电脑。

没有操作系统的电脑就是死机器，毫无用处。

windows有很多弊端，性能不稳定，收费，等等

有些项目要永久运行，那么就要求操作系统稳定运行，且不能关机

所以需要学习linux

好处如下：

  1. 稳定

  2. 开源，免费

  3. 多用户多任务

    同一时间段，windows只能同一时间同一个用户使用

windows 好处是用户量大，应用很多

linux主要就是用于开发

实际开发中，我们的服务器（高性能电脑，上面运行着项目）上大部分都是linux操作系统

虚拟机软件

这是一个软件，可以在本电脑上虚拟出一台电脑。这台虚拟的电脑就是一台裸机，可以在这个虚拟机软件里面装操作系统。

常用的虚拟机软件 VMWare，VirtualBox

虚拟机软件：就是一个能够虚拟出来计算机的软件。

虚拟机：被虚拟机软件虚拟出来的电脑。模拟一台真实的电脑。

学习Linux必须要知道的事

  • 千万细心
  • linux删除无法复原
  • 严格区分大小写
  • linux不需要拓展名
  • 没有盘符，所有文件都在/根目录之下
  • linux支持多用户同时操作

linux下重要的目录

/根目录：所有的文件都在这个目录之下

/home家目录：每个用户都会在/home里面有一个文件夹，这个文件夹就是这个用户的，每一个用户都会在这目录里有一个和自己的用户名同名的文件夹

/root:linux有一个超级用户，具备所有的权限，大部分情况下不建议直接使用root用户，所以ubuntu的设置就是让你创建一个用户

nosql

nosql-> not only sql 不仅仅是sql，只是mysql的辅助

当有海量数据的时候，MySQL有性能瓶颈，查询需要时间，查询条件越复杂，时间越长。

关系型和非关系型的区别

  • 关系型数据库大部分以表格的形式来存储，保存的是有关联的数据
  • 非关系型数据库，保存都是无关联的数据，数据的存储方式通常是以k-v键值对的形式
  • 关系型数据库适用于一些比较复杂的数据
  • 非关系型数据库适用于比较简单的数据
  • 关系型数据库支持事务，非关系型基本不支持事务
  • 关系型数据库有大概统一的一些规范，例如sql语句，非关系型不遵守sql规则，每一个产品都有自己的语法

非关系型数据库中的佼佼者就是 redis

Redis都有哪些特性

  • 性能极佳，查询数据的速度是110000次/s，写的速度是81000次/s
  • 丰富的数据类型
  • 可以做到持久化，不只是存在内存中，避免断电就丢失数据的现象

redis常用的使用场景

  • session共享
  • 热点数据，比如抢购

list相关的操作

就是一个列表，按照插入的顺序进行排序 （左边称为头部，右边称为尾部）

可以添加一个元素到list的头部或者尾部，这个list的底层实际上是个双向链表，对两端的操作性很高，通过索引取下标操作中间环节性能就差点。

zset有序集合

有序集合和无序集合很相似，没有重复元素。但是有序集合的每一个元素都有一个权重值，根据这个权重值进行排序，元素唯一，但是权重可以重复。 可以用于排行榜的制作。
